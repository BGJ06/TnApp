# Changelog

All notable changes to the **TN Party Connect** project are documented in this file.

## [1.0.0] - 2026-06-03

### Added
- **Core Architecture**: Integrated clean modular framework separating presentation, domain, data, and external service contracts.
- **Dynamic Styling**: Implemented responsive Material 3 palettes matching System Dark & Light themes using Google Outfit fonts.
- **Routing Module**: Designed static and dynamic route configuration mapping all dashboard interfaces.
- **Geographic Data Model**: Embedded Tamil Nadu Administrative boundaries (38 Districts & Taluks) inside a local JSON schema.
- **Authentication system**: Integrated OTP Mobile entry and dynamic leadership login suffix mapping.
- **Public access directory**: Enforced PII concealment controls permitting search of names and region scopes only.
- **Registration pipeline**: Built dynamic dropdown loaders validation workflow.
- **Emergency SOS dispatcher**: Implemented pulsing trigger and GPS coordinates listeners using Riverpod states.
- **Regional Approvals Dashboard**: Implemented status trackers for pending member profiles validation.
- **IT Wings capability matrices**: Formulated skill chips filters matching design, writing, and video capacities.
- **Backend Rules schema**: Formulated security configurations (`firestore.rules`) and compound query index bindings.
